Programa simple en Python que permite registrar alumnos con sus nombres, edades y notas. Muestra listados y estadísticas.

Cómo usar
1. Requisitos
Tener Python instalado en tu computadora

Un editor de texto o terminal

2. Ejecutar el programa
En Windows (CMD o PowerShell): python main.ipynb

3. Instrucciones paso a paso
Ejecuta el programa

Ingresa nombre del primer alumno y presiona ENTER

Ingresa edad y presiona ENTER

Ingresa nota y presiona ENTER

Responde si quieres ingresar otro alumno:

Escribe s y ENTER para continuar

Escribe n y ENTER para terminar
