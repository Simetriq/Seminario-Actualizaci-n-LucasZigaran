# Suma y Multiplicación de Matrices

Este proyecto permite sumar y multiplicar matrices usando una interfaz web.

## Archivos
- `server.js`: Servidor en Node.js (puerto 3000)
- `server.py`: Servidor en Python (puerto 5000)
- `index.html`: Interfaz web para ingresar matrices

## Cómo usar
1. Elige un servidor (Node.js o Python).
2. Ejecuta el servidor desde esta carpeta:
   - Para Node.js: `node server.js`
   - Para Python: `python server.py`
3. Abre el navegador en `http://localhost:3000` o `http://localhost:5000`.
4. Ingresa filas y columnas, genera las matrices, llena los valores y calcula.

## Requisitos
- Node.js para server.js
- Python para server.py
